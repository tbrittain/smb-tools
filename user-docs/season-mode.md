# Season Mode

smb-tools supports Super Mega Baseball 4's **Season Mode** saves, not just Franchise Mode. Setup and syncing work exactly the same way as Franchise Mode — see [Save Game Setup & Season Sync](save-game-setup.md) for the full walkthrough. This page covers what's different about tracking a Season Mode league.

## What Season Mode Is

Season Mode is a multi-season league, just like Franchise Mode: you play season after season, and smb-tools tracks standings, stats, and awards across all of them the same way. The difference is that **players and teams never evolve**. There's no aging, no attribute changes, no retirement, and no free agency — the player pool stays exactly as it started for the life of the league.

::: tip
Season Mode is not "a single season." You can sync and re-sync a Season Mode league across as many seasons as you play, exactly like Franchise Mode.
:::

## What's Different in smb-tools

- **Career stats still accumulate.** A Season Mode franchise builds up multi-season career stats and leaderboards identically to a Franchise Mode one.
- **The season selector works the same way.** Nothing about season navigation changes.
- **Hall of Fame is hidden.** Hall of Fame induction is based on career length and retirement — neither of which happens in Season Mode — so the Hall of Fame page and sidebar link don't appear for Season Mode franchises.
- **The player attribute chart may look flat.** Since attributes never change season to season in Season Mode, a player's attribute trend chart will show a flat line rather than a progression.

Everything else — syncing, snapshots, exports, awards, teams, and leaderboards — works the same regardless of which mode your save uses.

::: warning
Elimination Mode saves are still not supported.
:::
